Overview

The College Discovery Platform is a frontend web application designed to help students explore, search, and compare colleges in a structured and user-friendly interface. It provides essential college information in a clean and organized format to support better decision-making during the college selection process.

This project is built entirely on the frontend using modern web technologies.

Features
Search functionality to find colleges by name
Filter options based on location, courses, fees, and other criteria
College cards displaying summarized information
Detailed college information pages
College comparison feature to compare multiple institutions
Save feature to bookmark selected colleges for later viewing
Responsive user interface for mobile, tablet, and desktop devices

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
