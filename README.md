<h1>Overview</h1>

The College Discovery Platform is a frontend web application designed to help students explore, search, and compare colleges in a structured and user-friendly interface. It provides essential college information in a clean and organized format to support better decision-making during the college selection process.

This project is built entirely on the frontend using modern web technologies.

<h1>Features</h1>
<ul>
  <li>Search functionality to find colleges by name</li>
<li>Filter options based on location, courses, fees, and other criteria</li>
<li>College cards displaying summarized information</li>
<li>Detailed college information pages</li>
<li>College comparison feature to compare multiple institutions</li>
<li>Responsive user interface for mobile, tablet, and desktop devices
</li>

</ul>
